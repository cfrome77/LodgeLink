# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-08

### Added

#### Event Management
- Create and manage lodge events
- Configure event details and registration settings
- View attendee information in an organized dashboard

#### Registration
- Online attendee registration
- Administrative registration management
- Support for collecting attendee information

#### Event Check-In
- Fast attendee check-in workflow
- Real-time attendance tracking

#### Data Export
- Export attendee data to Excel (`.xlsx`)
- Export attendee data to CSV
- Configurable export columns

#### Security
- Secure authentication
- Protected administrative functionality
- Input validation and improved error handling

#### User Experience
- Responsive interface for desktop, tablet, and mobile devices
- Improved navigation and usability
- Optimized production deployment

### Notes

- Initial stable release of LodgeLink.
